## Upgrade Process

Follow these steps to upgrade MineHub safely.

<Steps>
  <Step title="Check Current Version" icon="info">

  In your dashboard, navigate to `Settings > About` to view your installed version.

  </Step>
  <Step title="Backup Data" icon="database">

  Export your projects and documents:

  ```bash
  minehub export --all --output=backup.json
  ```

  </Step>
  <Step title="Run Upgrade" icon="download">

  <Tabs>
    <Tab title="CLI" icon="terminal">

    ```bash
    minehub upgrade --version latest
    ```

    </Tab>
    <Tab title="Dashboard" icon="monitor">

    Go to `Settings > Updates` and click `Upgrade Now`.

    </Tab>
  </Tabs>

  </Step>
  <Step title="Verify" icon="check-circle">

  Test key features like order creation and document exports.

  <Callout kind="tip">
  Review migration notes in the release Update above for breaking changes.
  </Callout>

  </Step>
</Steps>

<Expandable title="Detailed Migration Notes for v1.1.0" default-open="false">

If upgrading from v1.0.x, update your API calls:

```javascript
// Old
fetch('https://api.minehub.com/v1/orders', {
  headers: { 'Authorization': `Bearer ${YOUR_TOKEN}` }
});

// New
fetch('https://api.minehub.com/v1/orders', {
  headers: { 
    'Authorization': `Bearer ${YOUR_TOKEN}`,
    'X-API-Version': '2'
  }
});
```

</Expandable>
