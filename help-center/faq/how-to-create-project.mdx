---
title: How do I create a new project?
description: Guide for starting a new project in MineHub.
---

## Steps

1. Sign in to your MineHub account.
2. On the dashboard, click "New Project".
3. Enter a name and optional description.
4. Click "Create".
5. Invite teammates if needed.

## Tips

- Project names must be unique.
- You can always edit project details later.
